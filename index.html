Een klein stukje over model wedstrijdvliegen

Modelvliegen is een vliegsport die vaak als recreatief gevlogen wordt. Ik vlieg zelf wedstrijden F3A-x/ IMAC. In deze klasse vlieg je kunstvlucht met grote modelvliegtuigen met een spanwijdte meer dan 2 meter zoals je het op de foto kunt zien. Het model waar ik mee vlieg is een Extra 330sc van krill-model schaal 1.3. Het model wordt casco geleverd en is geheel vervaardigd van composiet. Het voordeel van composiet is dat het model ontzettend stijf is, maar het nadeel is dat het model wat aan de zware kant is, zo'n 14,5kg bij een spanwijdte van 2.70m. Wel is het zo dat het model bij matige wind lekker strak vliegt.Nog een voordeel bij een groot model is dat je bij zo'n groot model met het opbouwen overal lekker erbij kan komen.

Instrumentarium op schaal 1.3 nagemaakt

Tijdens het bouwen van mijn Extra kreeg ik de inspiratie om de cockpit van de Extra voor een deel na te bouwen. Ik heb op internet gekeken welk type instrumentarium past bij mijn Extra. Via internet heb ik bij een speciaal zaak de benodigde instrumenten besteld. En heb ik later de cockpitinstrumentarium met LED verlichting van de Extra 330sc voor een deel nagebouwd, maakt het gehele model wel op schaal natuurgetrouw.
